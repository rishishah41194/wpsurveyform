== Description ==
WP Survey Form is the plugin which helps you to take Survey from your customers/users. It is very helpful to know about the popularity of any product on WooCommerce.

Some examples to Surveys:

1. Blog Survey.
2. Product Survey.
3. Take personal reviews.


== Installation ==
1. Activated the plugin.
2. Create Survey Form from the "add_new_survey_form" admin page.
3. Copy the shortcode from "display_survey_form" admin page and past it on anywhere in the site.
